📋 How to Test

This page helps you verify that the Visibility component correctly detects AI bots vs. humans.

Method 1: Browser DevTools

  1. Open DevTools (Press F12)
  2. Open Command Palette (Ctrl+Shift+P or Cmd+Shift+P)
  3. Type "Show Network conditions" and press Enter
  4. Uncheck "Use browser default" under User agent
  5. Try different user agents:
    • GPTBot/1.0 (ChatGPT)
    • ClaudeBot/1.0 (Claude)
    • PerplexityBot/1.0 (Perplexity)
    • Mozilla/5.0 (Regular Browser)
  6. Refresh this page after each change

Method 2: Using curl

curl -H "User-Agent: GPTBot/1.0" http://localhost:3000/demo/test.html
curl -H "User-Agent: ClaudeBot/1.0" http://localhost:3000/demo/test.html
curl -H "User-Agent: Mozilla/5.0" http://localhost:3000/demo/test.html
